# Hunter CS Department Website Recreation

## Project Overview

This project involves re-creating the Hunter CS department's website from scratch using HTML and CSS. 

## Project Goals

- **Hands-On HTML/CSS Practice:** Gain practical experience in writing HTML and CSS code.
- **Web Page Structure:** Learn how to structure web pages, including navigation bars, footers, and content sections.
- **User Interaction:** Implement functional navigation links and interactive elements.
- **Responsiveness:** Ensure the website is responsive and accessible on various devices.

## Project Details

### Components Implemented:

1. **Navbar/Footer:**
   - Responsive navigation bar with a logo and links.
   - Footer with a CUNY logo and a link to the CUNY homepage.

2. **Homepage:**
   - Banner/image section with a scalable gif.
   - Main content section with a heading, paragraph, and a "Learn More" link.
   - Announcement section with relevant content and a mailto link.

3. **About Page:**
   - Page with a heading describing the about section.
   - Two unordered list sections with provided copy/text assets.

4. **Courses Page:**
   - Page with a heading describing the courses section.
   - Table listing at least 5 different courses with relevant data.

5. **Contact Page:**
   - Page with a heading describing the contact section.
   - Contact form with input fields for first name, last name, email, reason, and subject.
   - Placeholder texts for input fields and a submit button.
   - Validation to indicate required fields.

6. **Mobile Responsiveness:**
   - Website is responsive for mobile resolutions (640px or less).
   - Optimized display of the navbar and content for smaller screens.

## How to Run

1. Clone this repository to your local machine.
2. Open the HTML files in your preferred code editor.
3. Run the HTML files in a web browser to preview the website.
